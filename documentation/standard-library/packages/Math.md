# Math
The `math` package contains many classes and utility methods related to mathematics.

The `Math` class contains many mathematical functions and utilities.


---


## Numbers
The math package contains classes for special numbers that cannot be represented by primitive types.

**Classes :** `Fraction`, `Complex`.



## Functions
The package contains classes that represent functions or sequence, which can be evaluated.

**Classes :** `Function`, `Sequence`, `ScalarField`, `VectorField`.


## Vectors
The package contains classes representing vectors.

**Classes :** `Vector`, `IntVector`, `LongVector`, `FloatVector`, `DoubleVector`.


## Matrices
The package contains classes representing matrices.

**Classes :** `Matrix`, `IntMatrix`, `LongMatrix`, `FloatMatrix`, `DoubleMatrix`.
