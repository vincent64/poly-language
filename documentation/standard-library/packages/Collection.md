# Collection
The `collection` package contains many classes representing different kind of collections with their associated operations.

The base class of most classes in this package is the `Collection` class, as well as the `Deque` subclass.


---


## List
A list is an ordered collection of objects. It can be represented using various data structures.

**Classes :** `List`, `ArrayList`, `LinkedList`.


## Queue
A queue is a collection of objects representing a First-In First-Out (FIFO) data structure.

**Classes :** `Queue`, `ArrayQueue`, `LinkedQueue`.


## Stack
A stack is a collection of objects representing a Last-In First-Out (LIFO) data structure.

**Classes :** `Stack`, `ArrayStack`, `LinkedStack`.
